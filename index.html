Virtual SOLE

Self Organized Learning Environment or SOLE is the brain child of TED prize winner Sugata Mitra. the main idea is that children can learn any topic if they are given an inspiring, hard question - one for which there is no simple yes/no answer - have access to the internet and can collaborate in a safe manner by themselves.

We are adapting the idea for the time of pandemic and create a safe, virtual environment.

Learn more, watch Sugata's TED talk
